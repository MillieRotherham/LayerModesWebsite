What are Difference Modes?

Difference modes reveal or emphasize the differences between the colors or luminance values of two layers.

Difference

This layer mode subtracts either the base color from the top color or the top color from the base color, again depending on which has the greater brightness value. If you blend with white it will invert the colours of the bottom layer, and like Exclusion if you blend with black there will be no change. The main difference between this and Exclusion is that there is more contrast between the two, and this is obvious in the example images. With this blend mode similar colours cancel each other out and the result is black.

Exclusion

Looks at the color information for each colour and subtracts either the top color from the base color or the base color from the top color, depending on which has the greater brightness value. This layer mode has almost the same effect as Difference, however it has lower contrast as can be seen in the example image. If the top colour has white it will invert the base colours, and blending with black produces no change. If there are similar colours in this layer mode they cancel out to be grey.

Subtract

Subtract subtracts the blend color from the base color for each colour in the image. It will cause a substantial darkening of the image, with whites being darkened the most and already dark colours changing little. Colour wise, it will take out the colours of the top layer from the bottom layer. That can be seen here on the example where there is no more yellow in the picture. Blacks and whites on the top layer will not change the bottom layer.

Divide

This layer mode looks at each colour and divides the blend color from the base color. This means that overall, images will become lighter and change colour. This is the main difference between Subtract and Divide. Regarding the colours, it will take out the colours of the top layer from the bottom layer. That can be seen here on the example where there is no more yellow in the picture.
