What are Colour Modes?

Colour modes, also known as composite or component modes change how the colour of the top layer interacts with the bottom layer.

Hue

This will create a final image with the grayscale information and saturation of the base color and the hue of the top color. This the colour of the final image is changed to the top colour, while how vibrant and the tones of the original image are retained. As is seen in the example it looks simply like the image has been remade with the only colour being yellow. It is less stronger effect than the Colour layer mode.

Saturation

The result has the highlights, shadows and hue of the base layer, along with the saturation of the blend colour. This means that most things are kept on the base layer, but the more pure colour the top layer has the more vibrant become the colours of the final image. If the top layer is grey, white or black there will be no change. In this example you can see that the saturation has been greatly increased, because the top colour (original as seen on the left) is very saturated.

Colour

This will keep the colours and saturation of the top layer while using the shadows and highlights of the lower layer. It is useful for tinting images with different colours, as can be seen in this example. Another thing that it can be used for is colouring monchrome images. This is the most common use of the layer mode, and it is useful for that. Another layer mode that gives a similar, less vibrant effect is the Hue layer mode.

Luminosity

This blend mode gives the image the hue and saturation of the base colour, while retaining the luminince of the top colour. That means that it keeps the general lightness of the page, while changing the colour of the image. It is the inverse effect of the colour mode, which means if you switched the order of the top and bottom layers around and gave the top layer the colour mode, you would have the same effect.
